  [33X[0;0YThis  chapter  documents  additional  functions available for computation of

  [1X3.1 [33X[0;0YWeights[133X[101X
  
  [1X3.1-1 Weight[101X
  
  [33X[1;0Y[29X[2XWeight[102X( [3Xelt[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ya list of integers[133X
  
  [33X[0;0YThe weight of the given element [3Xelt[103X is calculated and returned.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XV:= WeylModule(3,[3,3],"A",2);[127X[104X
    [4X[28XV[ 3, 3 ][128X[104X
    [4X[25Xgap>[125X [27Xm:= MaximalVectors(V);[127X[104X
    [4X[28X[ 1*v0, y1*v0, y2*v0, y1^(2)*y2*v0, -1*y1*y2^(2)*v0+y2*y3*v0, [128X[104X
    [4X[28X  y1*y2*y3*v0+y1^(2)*y2^(2)*v0 ][128X[104X
    [4X[25Xgap>[125X [27XWeight(m[2]);[127X[104X
    [4X[28X[ 1, 4 ][128X[104X
    [4X[25Xgap>[125X [27XList(m,Weight);[127X[104X
    [4X[28X[ [ 3, 3 ], [ 1, 4 ], [ 4, 1 ], [ 0, 3 ], [ 3, 0 ], [ 1, 1 ] ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YNOTE.  The above trick of applying the [10XWeight[110X function across an entire list
  [3Xlst[103X  of  vectors, with the command [10XList(lst, Weight)[110X, is very useful in many
  situations. This capability is built in to the [10XList[110X function in [5XGAP[105X.[133X
  
  
  [1X3.2 [33X[0;0YCharacters[133X[101X
  
  [33X[0;0YWe  have  already  seen  the  function  [2XCharacter[102X ([14X2.3-2[114X), that computes the
  (formal)   character  of  a  given  Weyl  module,  quotient,  submodule,  or
  subquotient.  We  now  consider  some  additional  functions  for  computing
  characters.[133X
  
  
  [1X3.2-1 [33X[0;0YDecomposeCharacter[133X[101X
  
  [33X[1;0Y[29X[2XDecomposeCharacter[102X( [3Xch[103X, [3Xp[103X, [3Xtyp[103X, [3Xrk[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ya list (of simple highest weights and their multiplicities)[133X
  
  [33X[0;0YIf  [3Xch[103X is a given character (of some module) then this function computes the
  multiplicities   of   the  simple  characters  in  [3Xch[103X,  thus  obtaining  the
  decomposition  numbers  of  the  module. Here it is necessary to specify the
  characteristic  [3Xp[103X  and  root system (of type [3Xtyp[103X and rank [3Xrk[103X) for the simple
  characters. For instance, this can be used to decompose tensor products.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XV:= WeylModule(2,[2,0],"A",2);[127X[104X
    [4X[28XV[ 2, 0 ][128X[104X
    [4X[25Xgap>[125X [27Xch:= ProductCharacter(Character(V),Character(V));[127X[104X
    [4X[28X[ [ 4, 0 ], 1, [ 2, 1 ], 2, [ 3, -1 ], 2, [ 0, 2 ], 3, [ 1, 0 ], 4, [128X[104X
    [4X[28X  [ 2, -2 ], 3, [ -2, 3 ], 2, [ -1, 1 ], 4, [ 0, -1 ], 4, [ 1, -3 ], 2, [128X[104X
    [4X[28X  [ -4, 4 ], 1, [ -3, 2 ], 2, [ -2, 0 ], 3, [ -1, -2 ], 2, [ 0, -4 ], 1 ][128X[104X
    [4X[25Xgap>[125X [27XDecomposeCharacter(ch,2,"A",2);[127X[104X
    [4X[28X[ [ 4, 0 ], 1, [ 2, 1 ], 2, [ 0, 2 ], 3, [ 1, 0 ], 2 ][128X[104X
  [4X[32X[104X
  
  
  [1X3.2-2 [33X[0;0YDifferenceCharacter[133X[101X
  
  [33X[1;0Y[29X[2XDifferenceCharacter[102X( [3Xch1[103X, [3Xch2[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ya list (a character)[133X
  
  [33X[0;0YIf  [3Xch1[103X  and  [3Xch2[103X  are  given characters, this function returns their formal
  difference character. It is used in the definition of the [10XDecomposeCharacter[110X
  function.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XDifferenceCharacter(Character(V),Character(V));[127X[104X
    [4X[28X[  ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe empty list here implements the zero character.[133X
  
  
  [1X3.2-3 [33X[0;0YProductCharacter[133X[101X
  
  [33X[1;0Y[29X[2XProductCharacter[102X( [3Xch1[103X, [3Xch2[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ya list (a character)[133X
  
  [33X[0;0YReturns  the product character of its inputs [3Xch1[103X and [3Xch2[103X. If [3Xch1[103X and [3Xch2[103X are
  characters  of  modules then the output of this function is the character of
  the tensor product of the modules.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XV:= WeylModule(2,[2,0],"A",2);[127X[104X
    [4X[28XV[ 2, 0 ][128X[104X
    [4X[25Xgap>[125X [27Xch:= ProductCharacter(Character(V),Character(V));[127X[104X
    [4X[28X[ [ 4, 0 ], 1, [ 2, 1 ], 2, [ 3, -1 ], 2, [ 0, 2 ], 3, [ 1, 0 ], 4, [128X[104X
    [4X[28X  [ 2, -2 ], 3, [ -2, 3 ], 2, [ -1, 1 ], 4, [ 0, -1 ], 4, [ 1, -3 ], 2, [128X[104X
    [4X[28X  [ -4, 4 ], 1, [ -3, 2 ], 2, [ -2, 0 ], 3, [ -1, -2 ], 2, [ 0, -4 ], 1 ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YBy  applying the function [2XDecomposeCharacter[102X ([14X3.2-1[114X) we can decompose tensor
  products in positive characteristic.[133X
  
  
  [1X3.2-4 [33X[0;0YSimpleCharacter[133X[101X
  
  [33X[1;0Y[29X[2XSimpleCharacter[102X( [3Xp[103X, [3Xwt[103X, [3Xtyp[103X, [3Xrk[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ya list (a character)[133X
  
  [33X[0;0YComputes  the simple character of highest weight [3Xwt[103X in characteristic [3Xp[103X. The
  arguments  [3Xtyp[103X  and  [3Xrk[103X  specify  the  type  and rank of the underlying root
  system. The function uses Steinberg's tensor product theorem.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XSimpleCharacter(2,[2,0],"A",2);[127X[104X
    [4X[28X[ [ 2, 0 ], 1, [ -2, 2 ], 1, [ 0, -2 ], 1 ][128X[104X
  [4X[32X[104X
  
  [33X[0;0YAnother  way  to  compute the same result is to compute the [10XCharacter[110X of the
  output  of  [10XSimpleQuotient(V)[110X,  where  [10XV[110X  is  the  WeylModule  in  the  same
  characteristic and root system with the same highest weight.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XV:= WeylModule(2,[2,0],"A",2);[127X[104X
    [4X[28XV[ 2, 0 ][128X[104X
    [4X[25Xgap>[125X [27XCharacter(SimpleQuotient(V));[127X[104X
    [4X[28X[ [ 2, 0 ], 1, [ -2, 2 ], 1, [ 0, -2 ], 1 ][128X[104X
  [4X[32X[104X
